Welcome to OMYM's documentation!
================================

OMYM (Organize My Music) is a powerful tool for organizing your music library based on metadata.

Features
--------

* Automatic music file organization based on metadata
* Support for multiple audio formats
* Customizable file naming and directory structure
* Preview mode for safe organization
* Robust error handling and logging
* SQLite database for tracking file operations
