Quick Overview
------------

OMYM helps you organize your music files by:

1. Analyzing metadata (artist, album, track numbers, etc.)
2. Creating a consistent directory structure
3. Renaming files according to your preferences
4. Handling special cases (multi-disc albums, compilations)

Basic Usage
---------

The basic workflow is:

1. Preview changes:

   .. code-block:: bash

       omym preview /path/to/music

2. Organize files:

   .. code-block:: bash

       omym organize /path/to/music

Features
-------

- Automatic organization based on metadata
- Preview mode for safe testing
- Support for multiple audio formats
- Customizable naming patterns
- Special character handling
- Multi-language support
